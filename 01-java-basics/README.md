# Topic 1 – Java Basics

## What was covered

- Statements are complete commands for Java to execute.
- Variables are named memory locations created by the user to store data.
- Keywords are reserved words in Java and cannot be used freely as custom names.
- Basic error handling was introduced.
- Primitive types:
  - `byte`
  - `short`
  - `int`
  - `long`
  - `double`
  - `float`
  - `char`
  - `boolean`
- `String` is not a primitive type. It is a class, but it is used in a very simple way, similar to primitives for beginners.
- Overflow and underflow happen when a value goes above the maximum limit or below the minimum limit of a type.
- Wrapper classes are class versions of primitive types.
- Casting is used when converting one type into another.
- Operators:
  - `=`
  - `-`
  - `+`
  - `*`
  - `/`
  - `%`
- Operands are the values used with operators.
- Common abbreviations:
  - `++`
  - `--`
  - `-=`
  - `+=`

## Challenges

- Print and modify `Hello World`
- Variable declarations
- Casting
- Simple calculations
